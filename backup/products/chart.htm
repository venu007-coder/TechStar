A platform independent web based 3D chart generating tool that can be customized for better graphic visualization, also designed to integrate with any J2EE application
ChartWeaver is a platform independent web based 3D chart generating tool-sending streams of interactive SVG (Scalable Vector Graphics) content to the browser in response to enterprise business data for graphic visualization.

Charts may be customized by the GUI based Chart Designer tool or ChartWeaver APIs. The clustering and monitoring services offer scalability and minimize server failure.

ChartWeaver is an on the fly chart generating tool, designed to integrate with any J2EE application. The application may be integrated with any enterprise Java based Web Application. ChartWeaver is most useful software solution for companies and developers looking for instant chart generation.
